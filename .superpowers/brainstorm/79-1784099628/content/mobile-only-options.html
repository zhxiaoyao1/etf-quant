纯手机运行方案

计算、存储、分析全部在手机上完成，无需外部服务器

A

增强型 PWA ⭐ 推荐

一个网页应用，但利用现代浏览器能力做到接近原生体验。

能力清单

  • IndexedDB — 本地数据库，存行情、信号、回测结果，永不过期
  • Web Workers — 后台线程跑量化计算，不卡界面
  • Service Worker — 离线缓存 + 后台定时拉取数据
  • OPFS — 私有文件系统，存模型参数、历史快照
  • 添加到主屏幕 = 全屏体验，和 App 一样

劣势

  • 超大规模回测（>500 只标的）会吃力
  • iOS 上后台任务有时间限制
B

React Native / Flutter 原生 App

优势

  • 真正的后台运行，不受浏览器沙箱限制
  • 可调用原生 API（通知、Widget 等）
  • 性能上限更高

劣势

  • 开发量是 PWA 的 2-3 倍
  • iOS 和 Android 需要分别适配和上架
  • 更新需要发版审核，不能像网页一样即时更新
  • 对个人项目来说维护成本太高
